# QORM 项目配色方案

## 1. 主色调

- **Primary Color** (主色):  
  `#111111`  
  说明：深黑色，稳重、简洁，适合品牌核心标志。

- **Accent Color** (强调色):  
  `#FF4F5C`  
  说明：柔和的红色，突出重要信息和动作按钮。

## 2. 背景色

- **Background Color** (背景色):  
  `#F9F9F9`  
  说明：明亮的灰色，营造干净、简约的感觉。

- **Secondary Background** (次要背景):  
  `#FFFFFF`  
  说明：白色，适用于卡片、表单背景等。

- **Dark Background** (深色背景):  
  `#1D1D1D`  
  说明：深灰背景，用于深色模式或页面的突出区块。

## 3. 文本色

- **Primary Text Color** (主要文本色):  
  `#333333`  
  说明：深灰色，适合正文、标题。

- **Secondary Text Color** (次要文本色):  
  `#777777`  
  说明：浅灰色，适合注释、次要信息。

- **Heading Text Color** (标题色):  
  `#000000`  
  说明：黑色，用于标题。

## 4. 按钮 & 交互元素

- **Primary Button** (主要按钮):  
  `#FF4F5C`  
  说明：鲜明的红色按钮，带有活跃感。

- **Secondary Button** (次要按钮):  
  `#2C3E50`  
  说明：深蓝灰色，用于副按钮。

- **Hover Effect** (悬停效果):  
  `#B5374D`  
  说明：按下时变为深红色。

## 5. 辅助色

- **Success Color** (成功色):  
  `#2ECC71`  
  说明：明亮的绿色，用于成功消息。

- **Error Color** (错误色):  
  `#E74C3C`  
  说明：红色，用于错误或警告。

- **Warning Color** (警告色):  
  `#F39C12`  
  说明：橙色，用于警告提示。

## 6. 边框 & 分隔符

- **Border Color** (边框色):  
  `#D1D1D1`  
  说明：浅灰色，用于边框和分隔线。

- **Divider Color** (分隔符色):  
  `#E1E1E1`  
  说明：非常浅的灰色，用于页面元素间的分隔。

## 使用场景

### UI界面

- 使用 **Primary Color** 和 **Accent Color** 进行按钮和关键标识的突出。
- 背景使用 **Background Color**，辅以 **Secondary Background** 和 **Dark Background** 形成层次感。
- 文本部分 **Primary Text Color** 和 **Secondary Text Color** 的组合可以保证良好的可读性。

### 警告与反馈

- **Error Color** 用于提示错误信息，**Success Color** 用于显示成功操作。
- **Warning Color** 用于显示警告信息，通常是需要用户注意的内容。

---

## 配色原则

1. **简洁统一**：主色调和辅助色的配合确保页面简洁，色彩统一。
2. **层次感清晰**：通过不同背景色和文本色，帮助用户在界面中快速定位和操作。
3. **强调重要信息**：通过高对比色（如 **Accent Color**）强调关键交互元素和信息。
