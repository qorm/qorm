# QORM Schemas

本目录用于存放 QORM JSON Schema。

建议文件：

```text
qorm.schema.json
scene.schema.json
component.schema.json
style.schema.json
action.schema.json
motion.schema.json
resource.schema.json
platform.schema.json
agent.schema.json
patch.schema.json
bundle.schema.json
```

Schema 应与 `docs/spec/json-format-spec.md` 保持一致。
